Tracking integration setup

Issue ingest keys for telematics platforms, custom middleware, and hardware forwarders. Each key is tied to a vehicle so live trip maps update when that asset is in transit. Use the key in the ingestKey field of the telemetry ingest API.

menu_book View integration documentation

How integrator keys work

  • link Link each key to an own-fleet vehicle — required for GPS ingest to attach to active trips.
  • vpn_key The ingest key is shown once at creation; store it in your integrator config or secret manager.
  • http POST telemetry to /ldms-trip-tracking/v1/system/telemetry/ingest with the key in the JSON body.
  • hub MQTT devices receive a broker topic after install — see the Connect trackers section.

Mobile driver apps and fuel sensors can be registered via Device installation or the advanced wizard above.
