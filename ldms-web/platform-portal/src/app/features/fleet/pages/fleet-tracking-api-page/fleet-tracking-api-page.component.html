Get live in three steps

LDMS uses a single REST ingest endpoint. Your device authenticates with an ingest key — no JWT, no API keys dashboard.

  1. 1
    Install & link a device

    Register the phone or hardware on Device installation and link it to a vehicle. GPS ingest requires a vehicle link.

  2. 2
    Copy your ingest key

    Shown once after install. Store it securely in your app config or device firmware — it identifies this tracker on the platform.

  3. 3
    POST telemetry

    Send latitude, longitude, and optional speed/heading every few seconds. When a trip is IN_TRANSIT for that vehicle, the live map updates instantly.
