Inventory management integration

Connect your ERP or WMS when you chose Sync via API. Issue API keys here so your external system can push products, categories, warehouse locations, stock levels, and transfers into LDMS. LDMS runs the full inventory module on synced data.

menu_book View integration documentation

What you can sync

  • category Product categories and product master (SKU as idempotency key)
  • warehouse Warehouse and depot locations used in transfers
  • inventory On-hand, reserved, and unit cost per warehouse
  • sync_alt Warehouse-to-warehouse transfers with en-route stops

Full component map, sync phases, and workflows are on the Components tab of the integration documentation.
